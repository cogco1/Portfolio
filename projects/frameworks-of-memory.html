STUDIO WORK · DESIGN-LED COLLABORATION

Frameworks of Memory

A market as a framework for collective memory and civic exchange.作为集体记忆与市井交换之框架的市场。

YearSpring 2026
TypeMarket / mixed-use
SiteNew York
RoleDesign lead · collaborative
CollaboratorYuhe Wang
ProfessorRonald Didonno
AUTHORSHIP

Design lead. Kaiwen led the overall architectural direction and principal design decisions; the project was developed collaboratively with Yuhe Wang.

设计主导。刘铠闻主导整体建筑方向与核心设计判断，项目与 Yuhe Wang 合作完成。

Frameworks of Memory treats the market as civic infrastructure — a covered public room where commerce, gathering, and everyday ritual accumulate over time. A clear structural frame holds a flexible interior, so the building becomes a stable armature for the changing life and memory of the neighborhood.

Frameworks of Memory 把市场视作一种市政基础设施——一个有顶的公共大厅，让商业、聚集与日常仪式在时间中层层累积。清晰的结构框架容纳灵活的内部，使建筑成为承载社区不断变化的生活与记忆的稳定骨架。

The project is carried from massing and program through to technical resolution, with a full set of plans, sections, and details developed to construction-document standard.

项目从体量与功能一直推进到技术深化，并按施工图标准完成整套平面、剖面与节点大样。
